# rclone

[Back to application catalog](../app_catalog.md)

## Description

Rclone is a command line program to sync files and directories to and from various cloud storage providers

## Homepage

[https://rclone.org](https://rclone.org)

## Module

You can load the module by:

```bash
module load rclone
```

!!! note "Note for using `rclone`"
    Run `module spider rclone` beforehand to check if this version requires any prerequisite modules.
